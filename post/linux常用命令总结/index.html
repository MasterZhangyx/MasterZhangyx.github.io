Linux常用命令总结

May 14, 2019
1 min read

sudo

使用root权限进行命令操作，例如:使用root用户编辑配置文件

sudo vim .bash_profile

echo

输出变量值，例如输出$PATH的值

echo $PATH

配置环境变量($PATH,export)

  1. 输入下面命令
sudo vim .bash_profile

  1. 输入e进入编辑页面

  2. 输入i进行编辑

  3. 在PATH变量后面配置xxx，其中如果没有$PATH，需要写成以下方式

   PATH=$PATH:XXX

  1. 如果存在$PATH，则进行追加即可，多个变量使用：分割。

  2. 可以使用export 进行变量定义，例如

   export HOME=XXX/XXX
   PATH=$PATH:$HOME
