HomeBrew安装教程（mac版本）

HomeBrew是什么

安装过程

  1. Command+空格键打开聚焦搜索，输入ter，打开终端,输入下面命令
/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
  1. 回车，根据提示输入账号密码，直到提示完成安装
  2. 输入brew –version判断是否安装成功

安装小问题总结

如果输入brew –version提示bash: brew: command not found ，尝试配置环境变量，需要在.bash_profile中配置一下path变量。在终端输入cd回到跟路径，输入 sudo vim .bash_profile指令编辑 .bash_profile文件，在文件中添加export PATH=/usr/local/bin:$PATH；如果PATH变量之前已经定义过，只需在后面追加 :/usr/local/bin(注：多个path变量之间通过” : “进行分隔)。输入完成之后执行source .bash_profile指令让bash_profile文件立即生效即可。
